=== ZoomText Screen Magnifier === zoomtext

==== Magnification ====

:>>Alt+Insert/Delete<<: Toogle magnification on/off
:>>Ctrl<< + kolečko myši: Increase/decrease magnification
:>>Ctrl+Shift+V<<: Quick overview of the screen

==== Reading ====

:>>Alt+Shif+S<<: Toogle speech on/off
:>>Alt+Shift+M<<: Toogle mouse echo on/off
:>>Ctrl+Alt+S<<: Read selected text
:>>Ctrl+Num5<<: Read current line
:>>Ctrl+Shift+Num5<<: Read current paragraph
:>>Ctrl+Alt+k/r<<: Announce the heading for column/row of table

==== Movements ====

:>>Alt+Home/End/PgUp/PgDown<<: Move to the left, right, up and down

==== Others ====

:>>Alt+Pause<<: Pass through key (the next key will be passed directly to application)
:>>Ctrl+Alt+Shift+U<<: Update screen
:>>Ctrl+Alt+Shift+T<<: Toogle cursor tracking on/off

== Applications == applications

This guide does not describe how to use the various native MS Windows
applications as this is sufficiently described in other guides. We however
point out the several Open Source applications which are very powerful and can
be used on MS Windows as well as on GNU/Linux.

=== Firefox === firefox

Firefox is a freely distributed internet browser. You can find
more information on [http://www.mozilla.com/firefox]

Using firefox with JAWS:
[http://www.jaws-users.com/text/Internet/firefox.htm]

=== Thunderbird === thunderbird

Thunderbird is a freely distributed email and news reader. You can find more
information on [http://www.mozilla.com/thunderbird]

Using Thunderbird with JAWS:
[http://www.jaws-users.com/scripts/ThunderBird/Thunderbird.htm]

=== OpenOffice.org === openoffice

OpenOffice.org is a freely distributable office suite, including a word
processor, spreadsheet calculator and a presentation tool. You can find
more information on [http://www.openoffice.org]

Its usage should be very similar to the one on GNU/Linux:
[gnu-usage#creating-documents OpenOffice on GNU/Linux]
